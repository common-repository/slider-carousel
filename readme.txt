Just another filterable gallery slider and carousel plugin. Simple but flexible and extendible.

== Description ==

Slider Carousel WordPress Plugin works with any WordPress theme, without any conflict. By using Slider Carousel WordPress Plugin, you can create more than 50 possible sliders and carousels with available options like text color, font size, background image, background color etc. Slider Carousel WordPress Plugin also comes with ready to use portfolio plugin which provides grid based portfolios/galleries and also masonry galleries/portfolios (in premium version).

= How to Use ? =

Once you have successfully installed and activated Slider Carousel plugin, you'll see SC Slider post type created on left side in admin dashboard navigation screen. Create some of your desired slider posts as per your needs.

Then just go to page or post edit screen and click on GT button in the buttons tab on content panel to insert shortcode. You can also use <b>[sc_slider]</b> shortcode where ever you want. 

Note: That shortcode button (to insert shortcode in the editor screen) is also shown in below screenshots.

== Installation ==

1. Upload the entire `sc-slider` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'SC Slider' menu in your WordPress admin panel.
Where you can create posts and categories to show your stuff nicely over the internet on your website.
For basic usage, you can also have a look at the [plugin homepage](http://www.filterable-portfolio.com/).
